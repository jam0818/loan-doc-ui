# システム連携テスト チェックリスト

## テスト実行方法

```bash
# 全テストを実行
bun run test:e2e

# 特定のテストファイルを実行
bunx playwright test e2e/auth.spec.ts

# UIモードで実行（デバッグ用）
bunx playwright test --ui

# レポートを確認
bunx playwright show-report
```
